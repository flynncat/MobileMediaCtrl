# 外接盘媒体浏览与复制工具 — 需求冻结（PRD 摘要）

## 1. 产品定位

面向老年人：在 **Windows** 上连接 **U 盘** 或 **手机（MTP）** 后，以**时间线**为主浏览照片与视频，支持**预览、多选、复制**，并可将文件**自由拖拽**到桌面、资源管理器或本应用另一窗口所代表的文件夹。

## 2. 时间线

- **主排序键**：优先 EXIF `DateTimeOriginal`（拍摄时间）；缺失则使用文件系统 **最后修改时间**（`LastWriteTime`）。
- **分组层级**：按 **自然日**（本地时区）分组展示；组可折叠/展开（UI 实现为按日标题 + 其下缩略图网格）。
- **扩展名白名单**（可配置，默认值）：常见图片与视频扩展名（如 `.jpg` `.jpeg` `.png` `.heic` `.webp` `.mp4` `.mov` `.m4v` 等）。

## 3. 「当前文件夹」与拖放落地规则

- **App 内跨窗口拖放**：拖入目标窗口时，文件复制到该窗口的 **当前目标路径**（`CurrentDropTargetPath`）。
- **当前目标路径**来源：用户通过 **路径栏** 浏览本机文件夹后选定；默认可为「桌面」或「用户图片」等安全默认项。
- **时间线与路径一致性**：时间线视图可能跨多个源子文件夹；**不在 UI 层用「整屏推断」单一文件夹**。跨窗口落地**仅**认路径栏指向的目录，避免误拷。
- **拖到系统（Explorer / 桌面）**：使用标准 Shell 拖放；**默认语义为复制**（与资源管理器一致：拖放时按住右键可选菜单；左键拖放目标由系统决定）。产品文档中向用户说明：**建议拖放到文件夹即完成复制**。

## 4. 多实例

- **单进程、多窗口**：每检测到一个新的外接 **U 盘卷** 或 **MTP 设备会话**，打开一个新的媒体窗口（`MediaWindow`）。
- **同设备重插**：若设备 ID（卷唯一标识 / MTP PnP ID）与已有窗口绑定一致，可**复用**该窗口并刷新目录（实现阶段可先做「总是新开」再优化为复用）。

## 5. 范围与隐私

- 仅枚举 **当前会话绑定的设备根**（U 盘根路径或 MTP 存储根）下的媒体文件；**不**建立全盘索引、**不**默认扫描其它固定磁盘、**不上传**任何用户文件。

## 6. 错误与反馈

- **复制结果**：完成后展示成功/失败数量；失败项简述原因（权限、空间、设备断开等）。
- **空间不足**：复制前可选检查目标盘剩余空间（大批量时）；失败时明确提示。
- **MTP 中断**：提示重新连接；列表与缩略图任务可取消。
- **同名文件策略**：默认 **跳过并计数**；可选 **自动重命名**（后缀 `(1)`）；**覆盖**必须二次确认。

## 7. 无障碍与体验（与 M5 对齐）

- 默认较大字体与可点击区域；关键操作有明确文案，避免专业术语（对外显示「手机存储」而非 MTP）。
- Tab 焦点顺序合理；重要按钮具备可访问名称。
